## Usage

Build the app: 
```shell script
make build
```

Run migrations: 
```shell script
make migrate-db-up
```

Drop migrations (1): 
```shell script
make migrate-db-down
```

Show logs: 
```shell script
make logs
```

Jump into psql: 
```shell script
make psql
```
